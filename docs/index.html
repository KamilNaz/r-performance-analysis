The full interactive HTML dashboard will be available shortly after the first GitHub Actions workflow completes. This automated process renders the RMarkdown report with live visualizations.

Report Features:

  • Real-time performance metrics and trends
  • Interactive Plotly visualizations
  • Response time and throughput analysis
  • Resource utilization monitoring
  • Anomaly detection and highlighting
  • Statistical correlation analysis
  • Hourly performance patterns
  • Actionable optimization recommendations

Quick Start (Local):

  • Clone the repository
  • Install R packages: install.packages(c('rmarkdown', 'ggplot2', 'dplyr', 'plotly', 'DT', 'knitr'))
  • Run: Rscript render_report.R
  • Open: docs/index.html in your browser
